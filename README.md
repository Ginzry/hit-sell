<a name="依赖包"></a>
### 依赖包
	* express : 基于 Node.js 平台的 web 应用开发框架
    * mongoose: 与MongoDB数据库进行交互的框架,需提前安装好mongodb并开启mongodb服务
    * morgan: 将程序请求过程的信息显示在Terminal中，方便调试
    * passport: 权限验证库
    * bcryptjs: 对用户密码进行hash加密。bcrypt的优化版，不需要安装任何依赖
